# app-fit-backend

## Requisitos

Para desenvolver e executar **app-fit-backend** localmente, tenha instalado:

- **Node.js** (v14 ou superior)

## Instalação e Execução

```bash
git clone https://github.com/MaschallEduardo/app-fit-backend.git
cd app-fit-backend
```

Instalar dependências
```Bash
npm install
#ou
yarn install
```

Rodar servidor
```Bash
node index.js
```
